# Getting Started

Welcome to Poodle! This guide will help you get up and running in minutes.

## Overview

Poodle is designed to be easy to use and integrate into your existing React projects. Whether you're building a new application or adding to an existing one, Poodle makes it simple.

## Prerequisites

Before you begin, make sure you have:

- **Node.js** 18.17 or higher
- **React** 18.0 or higher
- A package manager: **npm**, **yarn**, or **pnpm**

## Installation Steps

1. [Install Poodle](/docs/getting-started/installation) - Add Poodle to your project
2. [Quick Start](/docs/getting-started/quick-start) - Create your first component
3. [Configuration](/docs/getting-started/configuration) - Customize Poodle for your needs

## What's Next?
