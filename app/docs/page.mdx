# Introduction

Welcome to Poodle documentation! This guide will help you get started with our React component library.

## What is Poodle?

Poodle is a modern, lightweight React component library built with TypeScript and Tailwind CSS. It provides a collection of beautiful, accessible, and customizable components for your next project.

## Features

- 🎨 **Beautiful Design** - Clean and modern UI components
- ♿ **Accessible** - Built with accessibility in mind
- 🎯 **Type-Safe** - Full TypeScript support
- 🎨 **Customizable** - Easy to theme and customize
- 📦 **Tree-shakeable** - Import only what you need
- 🚀 **Performance** - Optimized for speed
